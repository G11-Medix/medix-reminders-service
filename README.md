# REMINDERS - Backend de Notificaciones

Backend en Node.js + TypeScript para envío de notificaciones:

- WhatsApp (Twilio)
- Push Notifications (Firebase)
- Integración con Supabase

---

## Tecnologías

- Node.js
- TypeScript
- Express
- Supabase
- Twilio
- Firebase

---


## Instalación

1. Clonar repo:
```bash
git clone <repo>
cd REMINDERS
```

2. Instalar dependencias:
```bash
npm install
```

---


## Ejecutar

Desarrollo:
```bash
npx ts-node-dev src/server.ts
```

Producción:
```bash
npm run build
node dist/server.js
```
