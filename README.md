# REMINDERS - Backend de Notificaciones


## Tecnologías

- Node.js
- TypeScript
- Express
- Supabase
- Twilio
- Firebase

---


## Instalación

1. Clonar repo:
```bash
git clone <repo>
cd REMINDERS
```

2. Instalar dependencias:
```bash
npm install
```

---


## Ejecutar



```bash
cd ../app-reminder
```

```bash
npx ts-node-dev src/server.ts
```

En dasarollo se debe usar NGrock y cambiar el cron en Supabase con la nueva direccion: 

```bash
ngrok http 3000
```
