# REMINDERS - Backend de Notificaciones


## Tecnologías

- Node.js
- TypeScript
- Express
- Supabase
- Twilio
- Firebase

---


## Instalación

1. Clonar repo:
```bash
git clone <repo>
cd REMINDERS
```

2. Instalar dependencias:
```bash
npm install
```

---


## Ejecutar

```bash
cd ../app-reminder
```

```bash
npx ts-node-dev src/server.ts
```
